# Menin
Code for analysis conducted during revisions of Sparbier, et al manuscript: https://www.nature.com/articles/s41556-022-01056-x
Experiments for which code is included in this repo are:

ChIPseq_181207: This is just reprocessing data used in a previous publication to be compatible with data from this project.

ChIPseq_220303: Chip for MLL1 in H9 and K562 cells w/ DMSO, EPZ, VTP & combo treatments, also K562 w/ CAS, MEN1-KO, EED-KO, & EED-POS_MEN1-KO.

ChIPseq_combined_200320_210422_210623: This is analysis combining 3 previously analysed MLL1 chip experiments including samples in K562 CAS, MENi, MEN-KO, LEDGF-KO, CAS, MEN1-KO, CAS-DMSO, CAS-VTP, KMT2Bc8-DMSO, & KMT2Bc8-VTP
